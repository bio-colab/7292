تمارين تفاعلية: الأعداد المركبة

مقدمة وأساسيات

الوحدة التخيلية $i$ تُعرّف بأنها $i = \sqrt{-1}$, وبالتالي $i^2 = -1$.

قوى $i$ تتكرر دورياً: $i^1=i$, $i^2=-1$, $i^3=-i$, $i^4=1$, $i^5=i$, ...

بشكل عام: $i^{4n+r} = i^r$ حيث $r$ هو باقي قسمة الأس على 4 ($r \in \{0, 1, 2, 3\}$).

العدد المركب يُكتب بالصيغة الجبرية $z = a + bi$, حيث $a$ هو الجزء الحقيقي و $b$ هو الجزء التخيلي ($a, b \in \mathbb{R}$).

تساوي عددين مركبين: $a_1 + b_1 i = a_2 + b_2 i$ إذا وفقط إذا كان $a_1 = a_2$ و $b_1 = b_2$.

كتابة الجذور السالبة: $\sqrt{-a} = \sqrt{a} i$ (لـ $a \ge 0$). مثال: $\sqrt{-9} = \sqrt{9}i = 3i$.

تمرين 1: بسّط $i^{35}$.

خطوات الحل:

نقسم الأس 35 على 4:

$35 \div 4 = 8$ والباقي $3$.

إذن $i^{35} = i^{4 \times 8 + 3} = i^3 = -i$.

تمرين 2: اكتب العدد $\sqrt{-12}$ بصيغة $bi$.

خطوات الحل:

$\sqrt{-12} = \sqrt{12} \sqrt{-1} = \sqrt{4 \times 3} i = 2\sqrt{3}i$.

يمكن كتابتها: $2\sqrt{3}i$ أو $2*sqrt(3)*i$.

تمرين 3: جد قيمتي $x, y$ الحقيقيتين إذا كان $(x+2) + 5i = 4 + (y-1)i$.

خطوات الحل:

بمساواة الأجزاء الحقيقية: $x+2 = 4 \implies x = 4-2 = 2$.

بمساواة الأجزاء التخيلية: $5 = y-1 \implies y = 5+1 = 6$.

$\therefore x=2, y=6$.

العمليات الجبرية

الجمع والطرح: $(a+bi) \pm (c+di) = (a \pm c) + (b \pm d)i$. (اجمع الحقيقي مع الحقيقي والتخيلي مع التخيلي).

الضرب: $(a+bi)(c+di) = (ac - bd) + (ad + bc)i$. (وزّع الأقواس وتذكر أن $i^2=-1$).

تمرين 4: جد ناتج $(5 - 2i) + (-3 + 7i)$ بالصيغة $a+bi$.

خطوات الحل:

$(5 - 2i) + (-3 + 7i) = (5 + (-3)) + (-2 + 7)i = 2 + 5i$.

تمرين 5: جد ناتج $(1+3i)(2-i)$ بالصيغة $a+bi$.

خطوات الحل:

$(1+3i)(2-i) = 1(2) + 1(-i) + 3i(2) + 3i(-i)$

$= 2 - i + 6i - 3i^2$

$= 2 + 5i - 3(-1)$

$= 2 + 5i + 3 = 5 + 5i$.

أو باستخدام الصيغة: $(1 \cdot 2 - 3 \cdot (-1)) + (1 \cdot (-1) + 3 \cdot 2)i = (2 + 3) + (-1 + 6)i = 5+5i$.

المرافق والقسمة

مرافق العدد: مرافق العدد $z=a+bi$ هو $\bar{z} = a-bi$. (غيّر إشارة الجزء التخيلي فقط).

خاصية هامة: $z \cdot \bar{z} = (a+bi)(a-bi) = a^2 + b^2$ (الناتج عدد حقيقي غير سالب).

القسمة: لقسمة $\frac{c_1}{c_2}$ (حيث $c_2 \neq 0$), نضرب البسط والمقام في مرافق المقام ($\bar{c_2}$):

$\frac{c_1}{c_2} = \frac{c_1}{c_2} \times \frac{\bar{c_2}}{\bar{c_2}} = \frac{c_1 \bar{c_2}}{c_2 \bar{c_2}} = \frac{c_1 \bar{c_2}}{|c_2|^2}$.

تمرين 6: ما هو مرافق العدد المركب $z = -4 + 9i$؟

خطوات الحل:

مرافق $z = a+bi$ هو $\bar{z} = a-bi$.

هنا $a=-4$ و $b=9$. المرافق هو $-4 - 9i$.

تمرين 7: ضع $\frac{3+i}{1-2i}$ بالصيغة $a+bi$.

خطوات الحل:

نضرب البسط والمقام في مرافق المقام $(1+2i)$:

$\frac{3+i}{1-2i} \times \frac{1+2i}{1+2i} = \frac{(3+i)(1+2i)}{(1-2i)(1+2i)}$

البسط: $(3+i)(1+2i) = 3(1) + 3(2i) + i(1) + i(2i) = 3 + 6i + i + 2i^2 = 3 + 7i - 2 = 1 + 7i$.

المقام: $1^2 + (-2)^2 = 1 + 4 = 5$.

الناتج: $\frac{1+7i}{5} = \frac{1}{5} + \frac{7}{5}i$. (أو $0.2 + 1.4i$).

الجذور التربيعية والمعادلات

الجذر التربيعي لعدد مركب: لإيجاد جذري العدد $c+di$, نفرض أن الجذر هو $x+yi$.

$(x+yi)^2 = c+di \implies (x^2-y^2) + (2xy)i = c+di$.

نحل المعادلتين: $x^2-y^2=c$ و $2xy=d$ لإيجاد قيم $x, y$ الحقيقية.

حل المعادلة التربيعية $ax^2+bx+c=0$ في $\mathbb{C}$: نستخدم الدستور (القانون العام):

$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$. إذا كان المميز ($b^2-4ac$) سالباً, فإن $\sqrt{b^2-4ac} = \sqrt{-(4ac-b^2)} = \sqrt{4ac-b^2} \, i$.

ملاحظة: إذا كانت معاملات المعادلة التربيعية $a,b,c$ حقيقية، فإن جذريها المركبين (إن وُجدا) يكونان مترافقين.

تمرين 8: جد الجذور التربيعية للعدد $3+4i$.

خطوات الحل:

نفرض الجذر $(x+yi)$. $(x+yi)^2 = 3+4i$.

$(x^2-y^2) + (2xy)i = 3+4i$.

(1) $x^2-y^2 = 3$

(2) $2xy = 4 \implies y = \frac{4}{2x} = \frac{2}{x}$.

نعوض (2) في (1): $x^2 - (\frac{2}{x})^2 = 3 \implies x^2 - \frac{4}{x^2} = 3$.

نضرب في $x^2$: $x^4 - 4 = 3x^2 \implies x^4 - 3x^2 - 4 = 0$.

نحلل: $(x^2-4)(x^2+1)=0$.

إما $x^2-4=0 \implies x^2=4 \implies x=\pm 2$.

أو $x^2+1=0 \implies x^2=-1$ (تهمل لأن $x$ حقيقي).

إذا $x=2$, $y=2/2=1$. الجذر الأول $2+i$.

إذا $x=-2$, $y=2/(-2)=-1$. الجذر الثاني $-2-i$.

الجذران هما $\pm(2+i)$.

تمرين 9: حل المعادلة $x^2 - 2x + 5 = 0$ في مجموعة الأعداد المركبة $\mathbb{C}$.

خطوات الحل:

نستخدم الدستور $a=1, b=-2, c=5$.

$x = \frac{-(-2) \pm \sqrt{(-2)^2 - 4(1)(5)}}{2(1)}$

$x = \frac{2 \pm \sqrt{4 - 20}}{2} = \frac{2 \pm \sqrt{-16}}{2}$

$x = \frac{2 \pm \sqrt{16}i}{2} = \frac{2 \pm 4i}{2}$

$x = \frac{2}{2} \pm \frac{4i}{2} = 1 \pm 2i$.

الحلان هما $1+2i$ و $1-2i$.

الصيغة القطبية ومبرهنة ديموافر

الصيغة القطبية: $z = x+yi = r(\cos \theta + i \sin \theta)$

حيث $r = |z| = \sqrt{x^2+y^2}$ (المقياس).

و $\theta = \arg(z)$ (السعة) تُحدد من المعادلتين: $\cos \theta = \frac{x}{r}$ و $\sin \theta = \frac{y}{r}$, مع مراعاة الربع الذي تقع فيه النقطة $(x,y)$.

مبرهنة ديموافر (للأسس الصحيحة): $[r(\cos \theta + i \sin \theta)]^n = r^n (\cos(n\theta) + i \sin(n\theta))$ لأي عدد صحيح $n$.

إيجاد الجذور النونية (نتيجة ديموافر): الجذور النونية للعدد $c = R(\cos \phi + i \sin \phi)$ هي:

$z_k = \sqrt[n]{R} \left[ \cos\left(\frac{\phi + 2k\pi}{n}\right) + i \sin\left(\frac{\phi + 2k\pi}{n}\right) \right]$

حيث $k = 0, 1, 2, \dots, n-1$. (يوجد $n$ جذور مختلفة).

تمرين 10: عبّر عن العدد $z = 1 - i$ بالصيغة القطبية (استخدم القيمة الأساسية للسعة $0 \le \theta < 2\pi$).

خطوات الحل:

$z = 1-i \implies x=1, y=-1$.

المقياس $r = \sqrt{1^2 + (-1)^2} = \sqrt{1+1} = \sqrt{2}$.

$\cos \theta = \frac{x}{r} = \frac{1}{\sqrt{2}}$, $\sin \theta = \frac{y}{r} = \frac{-1}{\sqrt{2}}$.

بما أن $\cos$ موجب و $\sin$ سالب، فـ $\theta$ تقع في الربع الرابع.

زاوية الإسناد التي جيب تمامها وجيبها $\frac{1}{\sqrt{2}}$ هي $\frac{\pi}{4}$.

السعة في الربع الرابع: $\theta = 2\pi - \frac{\pi}{4} = \frac{8\pi - \pi}{4} = \frac{7\pi}{4}$.

الصيغة القطبية: $z = \sqrt{2} \left( \cos \frac{7\pi}{4} + i \sin \frac{7\pi}{4} \right)$.

ملاحظة: إجابات مثل $\sqrt{2}(\cos(-\pi/4)+i\sin(-\pi/4))$ مقبولة أيضاً لكن السؤال طلب القيمة الأساسية $0 \le \theta < 2\pi$.

تمرين 11: احسب $(1+\sqrt{3}i)^4$ باستخدام مبرهنة ديموافر.

خطوات الحل:

1. نضع $z = 1+\sqrt{3}i$ بالصيغة القطبية:

$r = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1+3} = \sqrt{4} = 2$.

$\cos \theta = 1/2$, $\sin \theta = \sqrt{3}/2$. $\implies \theta = \pi/3$ (الربع الأول).

$z = 2(\cos(\pi/3) + i \sin(\pi/3))$.

2. نطبق ديموافر للأس 4:

$z^4 = [2(\cos(\pi/3) + i \sin(\pi/3))]^4$

$= 2^4 (\cos(4 \cdot \pi/3) + i \sin(4 \cdot \pi/3))$

$= 16 (\cos(4\pi/3) + i \sin(4\pi/3))$.

3. نجد قيم $\cos$ و $\sin$ للزاوية $4\pi/3$ (تقع في الربع الثالث، زاوية الإسناد $\pi/3$):

$\cos(4\pi/3) = -\cos(\pi/3) = -1/2$.

$\sin(4\pi/3) = -\sin(\pi/3) = -\sqrt{3}/2$.

4. نعوض القيم:

$z^4 = 16 (-\frac{1}{2} + i (-\frac{\sqrt{3}}{2})) = 16(-\frac{1}{2} - i \frac{\sqrt{3}}{2})$

$= -8 - 8\sqrt{3}i$.

تمرين 12: جد الجذور التكعيبية للعدد $8i$.

خطوات الحل:

1. نضع $c = 8i$ بالصيغة القطبية:

$c = 0+8i$. $R = \sqrt{0^2+8^2} = 8$.

النقطة $(0, 8)$ تقع على المحور التخيلي الموجب، إذن $\phi = \pi/2$.

$c = 8(\cos(\pi/2) + i \sin(\pi/2))$.

2. نطبق صيغة الجذور النونية ($n=3$):

$z_k = \sqrt[3]{8} \left[ \cos\left(\frac{\pi/2 + 2k\pi}{3}\right) + i \sin\left(\frac{\pi/2 + 2k\pi}{3}\right) \right]$ لـ $k=0, 1, 2$.

$z_k = 2 \left[ \cos\left(\frac{\pi + 4k\pi}{6}\right) + i \sin\left(\frac{\pi + 4k\pi}{6}\right) \right]$.

3. نحسب الجذور لـ $k=0, 1, 2$:

k=0: $z_0 = 2(\cos(\pi/6) + i \sin(\pi/6)) = 2(\frac{\sqrt{3}}{2} + i \frac{1}{2}) = \sqrt{3} + i$.

k=1: $z_1 = 2(\cos(5\pi/6) + i \sin(5\pi/6)) = 2(-\frac{\sqrt{3}}{2} + i \frac{1}{2}) = -\sqrt{3} + i$.

k=2: $z_2 = 2(\cos(9\pi/6) + i \sin(9\pi/6)) = 2(\cos(3\pi/2) + i \sin(3\pi/2)) = 2(0 + i(-1)) = -2i$.

الجذور هي: $\sqrt{3}+i$, $-\sqrt{3}+i$, $-2i$.

التمثيل الهندسي (شكل أرجاند)

يمكن تمثيل العدد المركب $z=x+yi$ بالنقطة $P(x,y)$ في المستوي المركب (مستوي أرجاند)، حيث المحور الأفقي هو المحور الحقيقي (Re) والمحور الرأسي هو المحور التخيلي (Im).

يمكن أيضاً تمثيله كمتجه من نقطة الأصل $O(0,0)$ إلى النقطة $P(x,y)$.

شكل أرجاند يمثل العدد المركب $z = 3 + 2i$.
